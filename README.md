# RPi Quiz v2

RPI Quiz v2 is the 2nd attempt to create a quiz platform for the Raspberry Pi. Works only with PS3-wired Buzz controllers.
Stack: Nodejs and React

## Usage

```javascript
npm run build
sudo npm run startAll
```
